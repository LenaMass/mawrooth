<h1>🇸🇦 Mawrooth</h1>

<h3>

<hr>

<h1>About The Project</h1>

<p>
Mawrooth is an educational iOS game that helps users stay connected to Saudi traditions and cultural knowledge through interactive matching-card gameplay.
</p>

<p>
The app is designed especially for young adults who want a fun and engaging way to remember cultural facts, symbols, traditions, foods, clothing, and heritage-related information without relying on internet access.
</p>

<p>
Instead of traditional reading-based learning, the game uses memory mechanics and visual interaction to improve focus, engagement, and long-term retention.
</p>

<hr>

<h1>Features</h1>

<ul>
<li>Interactive matching card gameplay</li>
<li>Saudi cultural and heritage-focused content</li>
<li>Collective page containing all saved cultural cards</li>
<li>Fully offline experience</li>
<li>Designed to improve memory retention and engagement</li>
<li>Clean and modern iOS interface</li>
</ul>

<hr>

<h1>Goals</h1>

<p>The main goal of the project is to make learning Saudi heritage:</p>

<ul>
<li>More engaging</li>
<li>Easier to remember</li>
<li>Accessible offline</li>
<li>Fun for younger generations</li>
<li>Interactive instead of passive</li>
</ul>

<h1>Gameplay Concept</h1>

<ul>
<li>Players match cultural cards together while learning facts related to Saudi heritage and traditions.</li>
<li>Each successful match helps reinforce memory retention through repetition and interaction, creating a more engaging educational experience compared to traditional learning methods.</li>
</ul>
